# ✅ Complete Website SEO Checklist (2026)

# 1️⃣ Technical SEO

Basic things search engines must see.

✔ Use **HTTPS**
✔ Create `robots.txt`
✔ Create `sitemap.xml`
✔ Use clean URLs
✔ Avoid duplicate pages
✔ Use canonical tags
✔ Enable gzip/brotli compression
✔ Proper redirect rules (301/302)
✔ No broken links (404 errors)
✔ Mobile-friendly layout

Example:

```txt
robots.txt
User-agent: *
Allow: /
Sitemap: https://yourdomain.com/sitemap.xml
```

---

# 2️⃣ On-Page SEO

Every page must follow this.

✔ Unique **title tag**
✔ Meta description
✔ H1 tag (only one per page)
✔ Proper heading structure (H1 → H2 → H3)
✔ Internal links
✔ Optimized images
✔ Keyword in first paragraph
✔ Clean readable URLs

Example:

```
/blog/nextjs-seo-guide
```

NOT

```
/post?id=8732
```

---

# 3️⃣ Content SEO

Google and AI rank **useful content**, not just keywords.

✔ Helpful content
✔ Long-form articles (1000+ words)
✔ FAQ sections
✔ Topic clusters
✔ Internal linking
✔ Regular updates
✔ Avoid duplicate content

Content should answer **search intent**.

---

# 4️⃣ Structured Data (Schema Markup)

Helps search engines and AI understand your site.

Use **JSON-LD schema**.

Example:

```json
{
 "@context": "https://schema.org",
 "@type": "Person",
 "name": "Malaika Anisar",
 "jobTitle": "Full Stack Developer",
 "url": "https://malaikaa.space"
}
```

Common schemas:

✔ Person
✔ Organization
✔ Product
✔ Article
✔ FAQ
✔ Breadcrumb

---

# 5️⃣ Performance Optimization

Speed is a ranking factor.

Improve **Core Web Vitals**.

✔ LCP under 2.5s
✔ CLS under 0.1
✔ INP under 200ms
✔ Use CDN
✔ Image optimization
✔ Lazy loading
✔ Minify CSS/JS
✔ Reduce bundle size

Tools:

* Google PageSpeed Insights
* Lighthouse

---

# 6️⃣ Image SEO

Images can bring a lot of traffic.

✔ Use descriptive filenames
✔ Add ALT text
✔ Compress images
✔ Use WebP/AVIF
✔ Add image sitemap

Example:

```
nextjs-portfolio-design.webp
```

---

# 7️⃣ Mobile SEO

Google uses **mobile-first indexing**.

✔ Responsive design
✔ Touch-friendly buttons
✔ Fast mobile loading
✔ Avoid intrusive popups
✔ Proper viewport settings

---

# 8️⃣ Backlinks (Authority)

Backlinks are still one of the **strongest ranking signals**.

✔ Guest posts
✔ Developer portfolios
✔ Open-source contributions
✔ Directory listings
✔ Social sharing
✔ Community forums

Good platforms:

* GitHub
* LinkedIn
* Medium

---

# 9️⃣ Analytics and Monitoring

Track everything.

Install:

✔ Google Analytics
✔ Google Search Console

Monitor:

✔ keywords
✔ clicks
✔ impressions
✔ indexing errors

# 🔟 AI-Agent Optimization (New in 2025-2026)

Websites should also be readable by AI agents.

Add:

✔ `llms.txt` file
✔ clean APIs
✔ semantic HTML
✔ structured content
✔ AI-readable documentation

Example:

```
/llms.txt
```

```

Purpose: Developer portfolio
Content: Projects, skills, blog
```

This helps AI systems like:

* OpenAI

understand your site better.
