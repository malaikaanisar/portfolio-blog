

## 1. Overview (Home of Dashboard)

A quick summary page when you log in.

Show cards like:

* Total Blogs
* Total Comments
* Pending Comments
* Total Visitors
* Most Viewed Post
* Latest Comments

This gives you a quick snapshot of your site.

---

## 2. Blog Management

Even though content is in Notion, the dashboard can still control it.

Features:

* View all posts
* Publish / Unpublish post
* Mark post as Featured
* Add tags or categories
* Re-sync posts from Notion
* Schedule publishing

Optional:

* Edit SEO metadata (title, description, OG image)

---

## 3. Comment Moderation

Since you already added comments, this is very important.

Add:

* View all comments
* Delete comments
* Mark spam
* Reply to comments
* Filter by post

Extra useful:

* Auto hide comments with links
* Spam detection

---

## 4. Visitor Analytics

Very useful for a portfolio.

Show:

* Daily visitors
* Page views
* Most visited blog
* Traffic sources
* Visitor countries
* Device types

You can integrate:

* Google Analytics
* Plausible
* Umami



## 10. Security Settings

Very useful if dashboard is public.

Add:

* Admin login
* Change password

## 12. Performance Monitoring

Helpful for developers.

Show:

* Page load time
* API response times
* Error logs
* Failed requests

---

## 14. Draft Workspace

For ideas.

Features:

* Save blog ideas
* Notes
* Todo list for blog topics
